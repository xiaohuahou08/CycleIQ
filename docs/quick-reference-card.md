# Wheel Strategy Quick Reference Card（Wheel 策略速查卡）

> Printable one-page reference（可独立打印的一页速查）

---

## A. Decision Flow（决策流程）

```text
Start
  ↓
筛选标的（愿意长期持有 + 流动性充足）
  ↓
Sell CSP（卖出现金担保看跌期权）
  ↓
到期是否被行权？
  ├─ 否：保留权利金 → 继续 Sell CSP
  └─ 是：买入100股 → Sell CC（卖出备兑看涨期权）
                    ↓
            到期是否被行权？
              ├─ 否：保留权利金 → 继续 Sell CC
              └─ 是：卖出100股 → 回到 Sell CSP
```

---

## B. CSP 开仓检查清单（Cash-Secured Put）

- [ ] 标的筛选：基本面可接受，愿意长期持有
- [ ] 流动性检查：成交量/未平仓合约数充足，买卖价差可接受
- [ ] 行权价选择：OTM，常见 Delta 约 -0.2 ~ -0.3
- [ ] DTE 选择：优先 30–45 DTE
- [ ] 权利金评估：年化回报与风险匹配
- [ ] 资金检查：现金担保充足（行权价 × 100 × 合约数）

---

## C. CC 开仓检查清单（Covered Call）

- [ ] 持股确认：已有对应 100 股/张
- [ ] 行权价选择：OTM，常见 Delta 约 0.2 ~ 0.3
- [ ] DTE 选择：优先 30–45 DTE
- [ ] 权利金评估：权利金与潜在封顶收益权衡
- [ ] 事件风险：财报/宏观事件前是否降低仓位

---

## D. Roll 决策树（滚仓 vs 到期）

```text
临近到期（通常 <= 7 DTE）
  ↓
是否接近/进入实值（ITM）？
  ├─ 否：可让其到期，按原计划循环
  └─ 是：
      ↓
  是否仍看好标的且希望保留策略连续性？
      ├─ 否：接受被行权/卖股，按纪律退出
      └─ 是：Roll（平旧仓 + 开新仓）
              - 调整到期日（向后）
              - 调整行权价（更保守或更激进）
              - 目标：净收权利金或改善持仓结构
```

---

## E. Core Guardrails（核心风控规则）

1. 不做不愿长期持有的标的。  
2. 单一标的和单一到期日不过度集中。  
3. 出现超预期波动时优先控制风险，再追求收益。  
4. 严格执行预设规则（止损、减仓、滚仓条件）。
